ESP32 Serial Firmware Flasher

Step 1: Files Ready

✓ bootloader.bin (18 KB)
✓ partitions.bin (3 KB)
✓ firmware.bin (510 KB)

Step 2: Connect ESP32

Make sure your ESP32 is connected to COM5

Note: This tool works in Chrome, Edge, or Opera browsers only (requires WebSerial API)

Step 3: Flash Firmware

Click the button below. The flasher will automatically handle the reset sequence.

Manual Instructions (if the button doesn't work):

  1. Go to: https://web.esphome.io/
  2. Click "CONNECT" and select COM5
  3. Click "INSTALL" → "Manual installation"
  4. Upload the three files with these addresses:
    • bootloader.bin at address 0x1000
    • partitions.bin at address 0x8000
    • firmware.bin at address 0x10000
  5. Click "INSTALL" to flash

Alternative: Use ESPTool Web

If the above doesn't work, try: https://espressif.github.io/esptool-js/

Same process: Connect → Upload the 3 files with their addresses → Program

After Flashing:

The ESP32 will reboot and start sending serial data at 115200 baud.

You should see messages like:

Sliders|2048|2048|2048|2048|2048
MuteButtons|false|false
